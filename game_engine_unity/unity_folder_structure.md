* Folders Structure

-------------------------------------------------------------------

* Folders Structure

```
* Project Name
   * Assets (Contain all source code, images, sounds, ...)
      - Art
      - Prefabs
      - Scenes
      - Scripts
      - Editor
      - Gizmos
      - NGUI
      - Plugins
      - Standard Assets
      
   * Project Settings (settings of project)
      
   * Library (Unity generated auto, should be git ignore)
   * Temp (Unity generated auto, should be git ignore)
```
