* **ANDROID**

  * M : Object
  * V : XML
  * C : Activity
  * **V <-> C** : findViewById
  
* **IOS**

  * M : Object
  * V : XIB
  * C : ViewController
  * **V <-> C** : Drag/Drop
  
* **Unity**

  * M : Object
  * V : GameObject
  * C : MonoBehaviour
  * **V <-> C** : Drag/Drop
  
* **ReactNative**

  * M : Javascript Object
  * V : Component
  * VM : State (Binding Components/Views to State)
  * **V <-> C** : When State update => View auto updated
  
  * **Vue**

  * M : Javascript Object
  * V : Template
  * VM : Data (Binding Template/Views to Data)
  * **V <-> C** : When Data update => Template auto updated
